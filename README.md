# Quest-Auto-Complete


Vencord plugin that automatically completes Discord quests for you !

## What it does

This plugin handles Discord quests automatically when you accept them. It works with video quests, game quests, streaming quests, and activity quests. Just accept a quest and the plugin takes care of the rest.

## Installation

1. Download the plugin file
2. Put it in your Vencord plugins folder
3. Restart Discord
4. Enable it in Vencord settings

## Settings

You can configure the plugin from the Vencord settings menu or use the Quest Settings button that appears on the quests page.

**Main options:**
- Auto-start quests when you accept them
- Show notifications for progress updates
- Display a progress bar at the top of the screen
- Skip video quests instantly
- Adjust completion speed for video quests

## Warning

Using this plugin could violate Discord's Terms of Service and may result in account suspension. Use at your own risk.
