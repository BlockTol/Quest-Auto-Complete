# 🎮 Quest AutoComplete

A powerful Vencord plugin that automates Discord Quests effortlessly. Automatically spoof games, watch videos, or stream to earn rewards with a sleek, real-time tracking UI.

[![Discord](https://img.shields.io/badge/Discord-Join_Server-5865F2?style=for-the-badge&logo=discord&logoColor=white)](https://discord.gg/9ra6MwHTHy)

## Features

| Feature | Description |
|---------|-------------|
| **Game Spoofing** | Automatically spoofs the required game to complete gameplay quests. |
| **Video & Stream Quests** | Fully supports watching promotional videos and streaming quests. |
| **Live Progress Bar** | Real-time progress bar tracking right at the top of your screen. |
| **Auto-Resume** | Automatically resumes quest automation even if you reload Discord. |
| **Smart Update Checker** | Notifies you of new plugin updates seamlessly within Discord. |

---
### Included Capabilities
- Support for multiple quest types (Play, Video, Stream).
- Fully customizable UI (Change progress bar color, notification durations).
- Smooth entry/exit animations for progress notifications.
- Smart stacking (Handles multiple quests running simultaneously).
- Bypasses active Discord window requirements for video quests.

### How to Use
1. Go to your **Vencord Settings** -> **Plugins**.
2. Enable **QuestAutoComplete**.
3. Open the **Settings** gear icon next to the plugin to customize colors and notification times.
4. Navigate to your **Discord Quests** page.
5. You will see an **"Automate"** button next to supported quests. Click it, and let the plugin do the rest!

---
## Configuration
You can customize the following in the plugin settings:
- **Notification Duration:** How long the UI pills stay on screen.
- **Auto Resume:** Automatically continue quests after restarting/reloading Discord.
